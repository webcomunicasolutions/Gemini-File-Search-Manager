# Canal de comunicacion - Proyecto FixMe Malaga

Canal bidireccional entre el proyecto **Gemini File Search Manager** y el proyecto **FixMe Platform**.

## Como funciona

### Si eres del proyecto FixMe (gestioo_info):
1. Escribe tus preguntas en `preguntas.md` de esta carpeta
2. El compañero de gemini_rag las leera y respondera en `respuestas.md`

### Si eres del proyecto Gemini RAG:
1. Revisa `preguntas.md` por si hay preguntas nuevas
2. Responde en `respuestas.md`
3. Las instrucciones principales van en:
   `/home/webcomunica/proyectos/proyectos_clientes/fixmemalaga/gestioo_info/integracion_rag/`

## Archivos

- `preguntas.md` - Preguntas del proyecto FixMe
- `respuestas.md` - Respuestas del proyecto Gemini RAG
